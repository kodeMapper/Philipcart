About This Project

I created this Flipkart website clone as a way to implement and showcase my knowledge of HTML and CSS. Through this project, I aimed to replicate the design of the Flipkart homepage using only these fundamental web technologies. By doing so, I not only honed my skills in HTML and CSS but also gained a deeper understanding of how to structure and style a complex webpage. I decided to name my version "Filipcart" to give it a unique identity while maintaining the essence of the original website.
